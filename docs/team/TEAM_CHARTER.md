# Team SH Charter

## Purpose
Team SH is a personal software engineering lab designed to help SungHwan learn how cross-functional engineering work happens in practice while building useful CV and video-related tools.

## Current scope
- Practical project planning
- Architecture review
- Lightweight implementation
- Decision logging
- Learning-oriented collaboration

## Out of scope for v0.1
- Full autonomous multi-agent orchestration
- Large-scale CI/CD automation
- Complex cloud-native production systems
- More than 3 core roles

## Roles
### SungHwan
- Final decision maker
- Project owner
- Learner
- Approver of scope and design choices

### PM Agent
- Defines user problem and MVP scope

### Tech Lead Agent
- Defines architecture and trade-offs

### Software Engineer Agent
- Converts decisions into implementable steps

## Working style
- Keep things small
- One project at a time
- Write down decisions
- Prefer clarity over sophistication
